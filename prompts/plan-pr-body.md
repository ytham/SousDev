1. Click on the plan file in the **Files changed** tab
2. Click the **+** button next to any line to leave an inline comment
3. For the first comment, click **"Start a review"** (not "Add single comment")
4. Add all your comments, then click **"Submit review"** when done

If the agent listed questions in the plan, answer them as inline comments on those lines.
